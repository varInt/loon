GitHub: Um sistema de controle de versão on-line conhecido como "GIT" ele faz exatamente aquilo que tu leu sobre SNV no wikipedia, ou pelo menos que eu pedi pra vc ler :P Aqui
Também armazena os fontes on-line para para servir de servidor/repositório do projeto.

Você pode fazer o download ( git Clone = clonar o repositório para sua máquina ) do SourceTree que é um programa que faz o download de todas as alteraçãoes do projeto e joga para sua maquina assim como pega todas as suas alterações locais e joga para o servidor; Isto é, o Sourcetree faz as operações básicas do SVN update/merge/commit. Essencial para quem trabalha em projeto / Equipes

Após efetuar o cadastro no GitHub informar me para te adicionar ao projetinho como colaborador assim vc também pode efetuar alterações no projeto.
SourceTree: Depois de instalado abra o programa e vá direto em Clonar / Novo no botão superior Esquerdo,
Na aba Clonar Repositório, busque pelo ícone que parece um Globo da terra que se encontra a direita em cima.
ao clicar abrirá uma nova janela Clique em Adicionar / Editar Contas...
Vá em Adicionar
Serviço de Hospedagem selecione: GitHub
URL do Host: https://github.com/varInt/loon.git
Nome do Usuário: é o nome do seu usuário mesmo NÃO PÕE O EMAIL ¬¬
digita a senha, seleciona a pasta que irá salvar o projeto, ex: c:\xampp\htdocs\loonGit\
Após isso o SourceTree irá efetuar um checkout(download de todos os arquivos) do projeto para sua máquina.
Agora é só acessar no browser http://localhost/loonGit/
